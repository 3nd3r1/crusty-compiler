# Crusty Compiler

A compiler for Crusty, a Rust-like language defined in the [language spec](https://hy-compilers.github.io/spring-2026/language-spec/). Written in Rust for the [Compilers course](https://hy-compilers.github.io/spring-2026/) at University of Helsinki.

## Progress

- [x] Setup
- [x] Tokenizer
- [x] Parser
- [x] Interpreter
- [x] Type checker
- [ ] IR Generator
- [ ] Assembly Generator
- [ ] Analysis & optimization
- [ ] Function definitions
